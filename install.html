Overleaf → Claude

Drag this button to your bookmarks bar (one time only):

Overleaf → Claude

Don't see a bookmarks bar under the address bar? Press Cmd+Shift+B (Mac) or Ctrl+Shift+B (Windows/Linux) to show it, then drag.

Each time you want Claude to read comments

  1. Open the Overleaf project and open the Review panel (comment icon, top right) so the comments are visible on the page.
  2. Click the Overleaf → Claude bookmark.
  3. It expands every "show more", captures all comments plus visible tracked changes, and downloads overleaf_comments_<project>_<date>.json to Downloads.
  4. Using Claude Code? Say "done" and Claude reads the file from Downloads. Otherwise the file is plain JSON, open it in any editor or point any script or LLM at it.

Limits. Captures the file currently open and the "Current file" tab only, so click once per file. Resolved comments are excluded unless you turn on Show resolved comments (inbox icon in the Review panel header) first. Tracked-change capture is partial (only rendered lines): for the full set, use Review → Accept in Overleaf, which syncs the text through git. If a comment is flagged possiblyTruncated in the file, open that thread fully and run again. If Overleaf redesigns the review panel, the bookmarklet may need a patch: ask Claude.

Safety. Runs only when clicked, only in your open tab, reads only what is on the page, downloads locally, sends nothing anywhere, stores no credential.
