Overleaf → Claude

Each time you want Claude to read comments

  1. Open the Overleaf project and open the Review panel (comment icon, top right).
  2. Click the Overleaf → Claude bookmark.
  3. It expands every "show more", captures all comments plus visible tracked changes, and downloads overleaf_comments_<project>_<date>.json to Downloads.
  4. Tell Claude "done". Claude reads the JSON from Downloads.

Limits. Captures the file currently open, so click once per file (e.g. main.tex, then appendix_theory.tex). Resolved comments stay hidden unless you unhide them. Tracked-change capture is partial (only rendered lines): for the full set, use Review → Accept in Overleaf, which syncs the text through git. If Overleaf redesigns the review panel, the bookmarklet may need a patch: ask Claude.

Safety. Runs only when clicked, only in your open tab, reads only what is on the page, downloads locally, sends nothing anywhere, stores no credential.

Fallback if the bookmark does nothing
